# PROMPT PEMBUATAN WEBSITE UI/UX INTERAKTIF

Buatkan sebuah website edukasi modern dan responsif dengan tema **"Antar Ruang dan Waktu"** yang menjelaskan hubungan ruang dan waktu dalam sejarah serta perubahan suatu wilayah dari masa ke masa.

## Tujuan Website

Website digunakan sebagai media pembelajaran IPS/Geografi untuk menjelaskan konsep antar ruang dan waktu melalui studi kasus **Sejarah Gedung Sate dan Kawasan Sekitarnya di Kota Bandung**.

## Teknologi yang Digunakan

* HTML5
* CSS3
* JavaScript
* Bootstrap 5 atau Tailwind CSS
* LeafletJS untuk peta lokasi
* Responsive Design (mobile, tablet, laptop)

## Konsep Desain UI/UX

Buat desain yang:

* Modern, bersih, dan profesional
* Menggunakan warna dominan biru tua, putih, dan aksen emas
* Memiliki animasi ringan dan smooth scrolling
* Mudah digunakan oleh siswa dan guru
* Mobile-first dan responsif
* Navigasi sticky navbar
* Memiliki ikon yang relevan
* Menggunakan card design modern
* Font modern seperti Poppins atau Inter
